Parent confirmation flow — v2

The CTA no longer asserts confirmation — it opens the pre-filled re-enrollment packet (review → sign → submit). "Confirmed" only appears when the packet is actually submitted, so the card can never disagree with the packet system. The link is an authenticated magic link: no password, but signed, single-family, and expiring.
